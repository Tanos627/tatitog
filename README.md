# tatitog— личная фильмотека
Автор
** Салмиярова Татьяна **

Описание
GUI-приложение «Book Tracker» для управления списком прочитанных книг с возможностью фильтрации, сохранения данных в формате JSON и использования Git.

Возможности программы:
- Добавление фильмов с указанием жанра, названия и года выпуска

Просмотр всех фильмов в виде таблицы
Фильтрация фильмов по жанру
Фильтрация по году выпуска
Удаление фильмов из коллекции
Автоматическое сохранение данных в файл JSON
Проверка корректности ввода:
-Год выпуска: должно быть целым числом от 1900 до 2026. -Рейтинг: должно быть числом от 0 до 10 (можно использовать дробные числа).

Как получить API-ключ
Перейдите на exchangerate-api.com
Зарегистрируйтесь (бесплатный тариф — 1500 запросов в месяц)
После регистрации скопируйте API-ключ с панели управления
Вставьте ключ в файл main.py (строка API_KEY = "ваш_ключ")
